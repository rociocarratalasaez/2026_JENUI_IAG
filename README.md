# Motivando el uso consciente y crítico de las herramientas de IAG entre el estudiantado
Archivos asociados al trabajo titulado "Motivando el uso consciente y crítico de las herramientas de IAG entre el estudiantado", presentado en las XXXII Jornadas sobre la Enseñanza Universitaria de la Informática (JENUI 2026)

## Financiación

Este trabajo ha sido posible gracias a la financiación recibida en el Proyecto de Innovación Docente "3901354 - Fomento del pensamiento crítico en el uso de los LLM entre los estudiantes de grados TIC'' de la Universitat de València.
