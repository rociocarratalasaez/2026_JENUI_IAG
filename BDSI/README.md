# Archivo asociado a la actividad realizada en la asignatura "Bases de datos y sistemas de información".

Los estudiantes recibieron el enunciado que figura en el archivo "BDSI_enunciado", listo para ser proporcionado a ChatGPT (o la IA Generativa de referencia del alumnado) y obtener el diseño de un modelo de base de datos relacional que contemple todos los requisitos descritos en él.

Aplicando la rúbrica presentada en la raíz de este repositorio, se debatió en clase cada aspecto evaluado de las distintas soluciones proporcionadas por las herramientas de IAG (generalmente, ChatGPT).
