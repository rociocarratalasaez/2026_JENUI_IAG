# Archivos asociados a la actividad realizada en la asignatura "Entornos de Usuario".

Los estudiantes habían realizado previamente la práctica cuyo enunciado figura en el archivo "Enunciado_original_PR.pdf".

Para la actividad descrita en el artículo, los estudiantes recibieron la propuesta de solución a la práctica completa generada usando ChatGPT simplemente enviándole a la herramienta de IA Generativa (IAG) el enunciado mencionado. Esta solución está formada por tres archivos: "IAG_solucion_controller.pdf", "IAG_solucion_model.pdf", "IAG_solucion_view.pdf".

Tras usar la rúbrica disponible en la raíz de este repositorio para evaluar por parejas la solución proporcionada por ChatGPT, el alumnado debatió sobre las calificaciones asignadas en cada caso para cada aspecto.

Finalmente, a partir de las preguntas que figuran en "Preguntas_ChatGPT_EU_debate.pdf" y "Reflexion_ChatGPT_EU_debate.pdf", se guió un debate en el aula sobre el uso de la IAG.
